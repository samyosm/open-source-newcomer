# Open Source Newcomer Program

This repository tries to help developers gain the confidence to start contributing in open source codebases.

## How to start?

If you would like to participate, follow the following steps:

1. Fork this repository
2. Clone your forked repository locally
3. Add your username to the list below
4. Push your changes
5. Make a pull request

These are the basics behind all contributions on Github. For more details please see [the full guide](https://oss.samyos.me/contribute).
